# 自动化工作流初始化版

这是一个干净的自动化工作流初始化模板，包含完整的架构但不包含任何运行时数据。

## 目录结构

```
自动化工作流初始化版/
├── .trae/                        # 核心配置
│   ├── rules/                    # 项目规则
│   │   └── project_rules.md      # 规则文件
│   └── skills/                   # 技能定义
│       └── autonomous-agent/     # 核心引擎
│
└── 自动化工作流组件库/            # 运行时数据目录
    ├── config/                   # 配置文件
    │   └── skill-registry.json   # 技能注册表
    ├── memory/                   # 记忆存储
    ├── delivery/                 # 交付文档
    ├── knowledge/                # 知识库
    ├── workflows/                # 工作流定义
    ├── swarm/                    # Swarm配置
    ├── templates/                # 模板文件
    └── logs/                     # 日志文件
```

## 使用方法

1. 将此目录复制到新项目根目录
2. 运行初始化命令：
   ```bash
   python ./.trae/skills/autonomous-agent/agent.py init
   ```

## 核心能力

- **一句话完成项目**: 自动规划、执行、验证、交付
- **专业方向强化**: 长期记忆系统支持知识沉淀
- **全领域稳定**: 知识边界感知确保输出质量
- **自适应专精**: 基于历史使用调整技能评分
